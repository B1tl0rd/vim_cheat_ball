# VIM Cheat Ball

A nice desktop paper ornament with Vim commands fast references.

![Vim cheat ball](http://i.imgur.com/tPfSoAS.jpg)


## Using

- Print the file *small.pdf* or *large.pdf* (A4 format)
- Cut
- Fold
- Glue it
