VIM Cheat Ball
==============
A nice desktop paper ornament with Vim commands fast references.

![Vim cheat ball](http://i.imgur.com/tPfSoAS.jpg)



Using
-----

- Print the file *vimcheatball3.pdf* (A4 format)
- Cut following the marks
- Fold following the marks
- Glue it!
